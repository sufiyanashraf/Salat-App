1. Create `AladhanModels.kt`:
   - Use @Serializable data classes matching the Aladhan API v1 response:
   - AladhanResponse(code: Int, data: AladhanData)
   - AladhanData(timings: AladhanTimings, date: AladhanDate)
   - AladhanTimings(Fajr: String, Dhuhr: String, Asr: String, Maghrib: String, Isha: String)
   - AladhanDate(readable: String, gregorian: GregorianDate)
   - GregorianDate(date: String)  // "DD-MM-YYYY" format
   - For monthly: AladhanMonthResponse(code: Int, data: List<AladhanData>)

2. Create `AladhanApiService.kt`:
   - class AladhanApiService with HttpClient using OkHttp engine + ContentNegotiation with JSON (ignoreUnknownKeys = true)
   - suspend fun getTimingsByDate(date: String, latitude: Double, longitude: Double, method: Int = 2): AladhanResponse
     → GET "https://api.aladhan.com/v1/timings/{date}?latitude={lat}&longitude={lng}&method={method}"
   - suspend fun getTimingsForMonth(year: Int, month: Int, latitude: Double, longitude: Double, method: Int = 2): AladhanMonthResponse
     → GET "https://api.aladhan.com/v1/calendar/{year}/{month}?latitude={lat}&longitude={lng}&method={method}"
   - fun close() to close the HttpClient

3. Create `UserPreferences.kt`:
   - Object/class using DataStore<Preferences>
   - Keys: LATITUDE (doublePreferencesKey), LONGITUDE (doublePreferencesKey), IS_LOCKED (booleanPreferencesKey), LAST_PRAYER_NAME (stringPreferencesKey)
   - Extension property: Context.userPreferencesDataStore
   - Functions: getLocation() → Flow<Pair<Double, Double>?>, saveLocation(lat, lng), getIsLocked() → Flow<Boolean>, setIsLocked(Boolean)

4. Create `PrayerTimeRepository.kt`:
   - class PrayerTimeRepository(private val dao: PrayerTimeDao, private val api: AladhanApiService)
   - suspend fun fetchAndCacheTodayPrayerTimes(lat, lng): maps API response → PrayerTimeEntity → inserts into Room
   - suspend fun fetchAndCacheMonthlyPrayerTimes(year, month, lat, lng): fetches month → inserts all
   - fun getTodayPrayerTimes(lat, lng): Flow<PrayerTimeEntity?> (from Room)
   - Helper: convert AladhanTimings to PrayerTimeEntity, parsing "DD-MM-YYYY" to "yyyy-MM-dd"

5. Update `AndroidManifest.xml`:
   - Add <uses-permission android:name="android.permission.INTERNET" />
